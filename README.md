# ogit
## Install
```
git clone https://github.com/elviriljazi/ogit/
cd ogit
sudo ogit -install
```
## Manual
```
sudo ogit config
>set username
>set repositories
```
#### List repositories
```
ogit -l
```
#### Add new repository
```
sudo ogit -l add
```
## Use
```
ogit indexOfRepo baseBranch compareBranch
```
