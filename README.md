# ogit
## Install
```
wget https://raw.githubusercontent.com/elviriljazi/ogit/main/ogit
sh ogit -install
```
## Manual
```
sudo ogit config
>set username
>set repositories
```
#### List repositories
```
ogit -l
```
#### Add new repository
```
sudo ogit -l add
```
## Use
```
ogit indexOfRepo baseBranch compareBranch
```
```
ogit      2        2.1.11      2.1.12
```
